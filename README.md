# Sukkur IBA Shuttle System

This is an end semester project developed by **Yash Raj** and **Yasir**.

## Description

A web-based shuttle booking system for Sukkur IBA University, allowing students to view schedules, explore routes, and book shuttle rides conveniently.

## Features

- View shuttle schedules
- Explore available routes
- Book rides online
- Filter schedules by route

## Technologies Used

- HTML
- CSS
- JavaScript

## Project Structure

- `index.html` - Main HTML file
- `styles.css` - Custom CSS styles
- `js/` - JavaScript files for functionality
- `data/` - JSON data files for routes and schedules
- `images/` - Project images and logos
